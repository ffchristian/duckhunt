Simple emulador (demo) del clásico de Nintendo Duck Hunt usando Node, React, jQuery, JS, Gulp, Sass y Git.

La idea completa es hacer un duplicado en pantalla con Socket.io y usar un control de Nintendo en el celular para controla el sensor de disparo en la pantalla de escritorio.

Instalación local:

git clone https://github.com/christianirack/duckhunt

cd duckhunt

npm install

npm start
